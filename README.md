# Django-Pinterest


Pinterest 클론 코딩 개인 프로젝트

- Django + Html / Css / JavaScript를 이용한 풀스택 개발
- Class View기반으로 구성
- Django Form을 이용한 Html 간결화
- Gunicorn + NginX + Docker + AWS를 이용한 배포
- HTTPS 웹보안 작업 진행
- MariaDB를 이용한 데이터베이스 구축
